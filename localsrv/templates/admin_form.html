Quick start:

  1. Click "Add New" button and create new search base with a desired name;
  2. Open link of new base ({{ request.scheme }}://{{ request.get_host }} ...). Change doxygen project according instruction and generate doxy doc;
  3. Click "Update" button of the base. Select file with new search data (searchdata.xml) and click "Update". Wait for the message ... updated successfully .. or use another (right) file in case of error;
  4. Open your doxygen documentation and use search field.
