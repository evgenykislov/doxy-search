Quick start:

  1. Click "Adds new documentation" button and add new documentation with a desired name and empty path;
  2. Open link of new documentation (http://127.0.0 ...), change doxygen project according instruction and generate the documentation. Back to main page;
  3. Click "Change" button of the documentation and set path to searchdata.xml file was generated;
  4. Click "Update" button of the documentation. Wait for the message ... updated successfully .. or fix file path in case of errors;
  5. Open your doxygen documentation and use search field.
