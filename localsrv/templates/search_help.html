Help about usage:

Open Doxyfile in doxywizard and set the following fields in the Expert tab in the HTML topic:

SEARCHENGINEYes
SERVER_BASED_SEARCHYes
EXTERNAL_SEARCHYes
SEARCHENGINE_URLThis page url: {{ request.build_absolute_uri }}
SEARCHDATA_FILEsearchdata.xml

Run doxygen with new settings. You have to wait for the message *** Doxygen has finished

Find generated file searchdata.xml and copy its path.

Go to admin page /localsrv/admin
