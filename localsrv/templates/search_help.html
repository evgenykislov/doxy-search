Help about usage:

Open Doxyfile in doxywizard and set the following fields in the Expert tab in the HTML topic:

SEARCHENGINEYes
SERVER_BASED_SEARCHYes
EXTERNAL_SEARCHYes
SEARCHENGINE_URLPage url. Get from address bar.
SEARCHDATA_FILEsearchdata.xml

Run doxygen
